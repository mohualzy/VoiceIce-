## 摆线

1. **定义**

   当一个圆沿一条定直线作无滑动的滚动时，圆周上一个定点的轨迹称为摆线（也称旋轮线）<br><br>

2. **参数方程**

   设滚动圆的半径为 $a$，转过的角度为 $t$（视为参数），则摆线的参数方程为：

$$\Large \begin{cases} x = a(t - \sin t) \\ y = a(1 - \cos t) \end{cases}$$

3. **图片**<br>
   ![a = 0.58时的摆线](picture/Pasted%20image%2020251220024139.png)<br><br><br>
4. **常用性质**（一拱范围内 $0 \le t \le 2\pi$）<br><br>

   * **弧长**：$\Large s = 8a$<br><br>
   * **面积**：$\Large A = 3\pi a^2$<br><br>
   * **切线斜率**：$\Large\dfrac{dy}{dx} = \cot \dfrac{t}{2}$
<br><br>
## 心形线

1. **定义**

   当一个圆沿着另一个半径相同的圆的外侧滚动时，滚动圆上一点的轨迹
   <br>
2. **方程**

   - **极坐标方程**（最常用）：$\Large r = a(1 + \cos \theta)$ （此时尖点在原点，图形向右凸）<br><br>*变体*：$\Large r = a(1 - \cos \theta)$ （图形向左凸）<br><br>
   
   - **参数方程**：
    
    $$\Large \begin{cases} x = a(2\cos t - \cos 2t) \\ y = a(2\sin t - \sin 2t) \end{cases}$$
    <br>
3. **图片**<br><br>
   ![a=1时的心形线|400](picture/Pasted%20image%2020251220030152.png)
<br><br>
4. **常用性质**<br><br>
   - **周长**：极坐标形式下$\Large L = 8a$，参数方程形式$\Large L = 16a$<br><br>
   * **面积**：$\Large \dfrac{3}{2} \pi a^{2}$

## 星形线

1. **定义**

   一个半径为 $a/4$ 的小圆在半径为 $a$ 的大圆内侧无滑动滚动时，小圆周上任意一点的轨迹。<br><br>

2. **方程**

  -  **直角坐标方程**：$\Large x^{2/3} + y^{2/3} = a^{2/3}$<br><br>
  
  - **参数方程**（常用）：
$$\Large \begin{cases} x = a \cos^3 t \\ y = a \sin^3 t \end{cases}$$
    
3. **图片**<br><br>
   ![a=3.25时的星形线](picture/Pasted%20image%2020251220032809.png)<br><br>
4. **常用性质**
<br>
   - **全长**：$\Large L = 6a$<br><br>
   - **面积**：$\Large A = \dfrac{3}{8}\pi a^2$

## 伯努利双纽线 

1. **定义**

   设定点 $F_1, F_2$ 距离为 $2a$，动点 $M$ 满足 $|MF_1| \cdot |MF_2| = a^2$ 的轨迹。形状像 $\infty$<br><br>

2. **方程**<br><br>
   - **直角坐标方程**：$\Large (x^2 + y^2)^2 = 2a^2(x^2 - y^2)$<br><br>
   - **极坐标方程**：$\Large r^2 = 2a^2 \cos 2\theta$<br><br>
3. **图片**<br><br>
   ![a=3.46时的伯努利双纽线](picture/Pasted%20image%2020251220033223.png)
<br><br>
4. **常用性质**<br><br>
   - **对称性**：关于原点、x轴、y轴均对称。<br><br>
   - **面积**：利用极坐标积分计算，单圈面积为 $\Large a^2$，总面积为 $\Large 2a^2$。<br><br>
