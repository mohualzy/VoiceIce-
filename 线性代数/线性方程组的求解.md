
# 线性相关/无关

## 判断

1. 行列式法
   - det = 0  => 有关
   - det != 0  => 无关
2. 矩阵秩法
   - r < k  (k为向量个数) => 相关
   - r = k  => 无关
1. 定义法
   存在一组不全为0的ki，使得求和ki * ai = 0，则相关，否则无关

# 线性方程组

## 齐次

**方法一：高斯消元法（推荐）**

**步骤**：

1. 写出系数矩阵 A
    
2. 用初等行变换化为行最简形（RREF）
    
3. 确定主元列和自由变量列
    
4. 令自由变量为参数，表示主元变量
    
5. 写出通解形式
    

**例子**：  
求解方程组：

{x1+2x2+3x3=02x1+4x2+6x3=03x1+6x2+9x3=0⎩⎨⎧​x1​+2x2​+3x3​=02x1​+4x2​+6x3​=03x1​+6x2​+9x3​=0​

**解**：

1. 系数矩阵：
    

A=(123246369)A=​123​246​369​​

2. 行变换：
    

(123000000)​100​200​300​​

3. 主元列：第1列；自由变量：x₂, x₃
    
4. 令 x₂ = t, x₃ = s，则 x₁ = -2t - 3s
    
5. 通解：
    

(x1x2x3)=t(−210)+s(−301)​x1​x2​x3​​​=t​−210​​+s​−301​​

**方法二：基础解系法**

当 rank(A) = r < n 时：

- 基础解系包含 n - r 个线性无关的解向量
    
- 通解是这些向量的线性组合

## 非齐次

**方法：增广矩阵法**

**步骤**：

1. 写出增广矩阵 \[A | **b**]
    
2. 用初等行变换化为行最简形
    
3. 判断解的情况：
    
    - 无解：出现 0 = 非零 的行
        
    - 唯一解：rank(A) = rank(\[A|**b**]) = n
        
    - 无穷多解：rank(A) = rank(\[A|**b**]) < n
        
4. 求特解：令自由变量为0，解出主元变量
    
5. 求齐次通解：解 A x = **0**
    
6. 写出通解：特解 + 齐次通解

**注意：**

对于非齐次线性方程组 **Ax** = **b**：

- 设系数矩阵 **A** 的秩为 r
    
- 设增广矩阵\[**A** | **b**] 的秩为 r'
    

**解的判定**：

1. **如果 r ≠ r'**：方程组 **无解**
    
2. **如果 r = r'**：
    
    - 且 r = n（未知数个数）：**唯一解**
        
    - 且 r < n：**无穷多解**
